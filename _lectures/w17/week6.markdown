# File Redirection

## Operators

  - `<` send file as input
  - `>` send output to file (create/overwrite)
  - `>>` append output to file (create/preserve)
  - `>&` merge output
  - `<&` merge input
  - `<<` "Here Document"

## Descriptors, Name and Number

  - 0: Standard Input (stdin, cin)
  - 1: Standard Output (stdout, cout)
  - 2: Standard Error (stderr, cerr)
