touch

touch is used for changing file timestamps as well as creating new files. It's useful for keeping track of the last time you edited a file for easy access later.

$touch*  helloWorld.txt

Another Short Example

$touch text.txt 
$ ls -l text.txt
-rw-rw-r-- 1 [user] [user] 0 Mar 20 23:33 text.txt

By applying the touch command to text.txt, it updated the access and modification time of the file to the time that the command was used.

Useful Options / Examples

You'll never believe all the crazy, zany, fantabulously bonkers stuff you can do wth this command, but here's some of that stuff anyway:

touch -a

Changes only the access time. Useful if you want a timestamp but didn't change anything.

touch -c, --no-create

If the touch command is used with a filename not found in the current directory, a new file will be created by default. This command disables that feature, instructing touch not to create any files.

touch -d, --date=STRING

Parses STRING and uses it instead of the current time. Good for if you want to write a custom time or marker to reference a touch.
