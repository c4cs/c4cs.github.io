whoami
--
`whoami` prints the username of the current user when invoked
<!-- one line explanation would go here -->

<!-- minimal example -->

~~~ bash
$ whoami
mmdarden
~~~

<!--more-->

### Example Command
`whoami` returns root if you are the system administrator

~~~ bash
$ whoami
root
~~~

#### 
`whoami` has been obsoleted by the `id` command

~~~ bash
$ id -un
mmdarden
~~~
