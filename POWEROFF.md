powerofff
-------

poweroff instructs your system to power down.

~~~ bash
poweroff
~~~

### Examples

#### Example command
~~~ bash
poweroff
~~~

#### Example command (equivalent to previous)
~~~ bash
halt -p
~~~

#### Example command (equivalent to previous)
~~~ bash
halt --poweroff
~~~

### Important note
if you aren't logged in as the root user you wil need to use the sudo command:
~~~ bash
sudo poweroff
~~~
